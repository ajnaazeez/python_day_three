# Python Practice - Day 3 

This repository contains my Day 3 Python learning practice.  
The exercises mainly focus on control flow concepts like conditions and loops along with small logical programs.

##  Topics Covered
- Conditional Statements (if, elif, else)
- Nested if statements
- For loop
- While loop
- Range function
- Basic pattern programs

##  Practice Programs
Some of the tasks included:
- Positive, negative or zero number check
- Largest of three numbers
- Grade calculator
- Multiplication table
- Sum of natural numbers
- Number guessing program
- Simple star pattern printing

##  Purpose
The objective of this project is to improve logical thinking and understand how program flow works using conditions and loops in Python.
